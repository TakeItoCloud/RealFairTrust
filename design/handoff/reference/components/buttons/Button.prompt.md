The primary call-to-action — calm gold gradient pill on the navy stage; use it for the single most important action on a view.

```jsx
<Button variant="primary" size="lg">Encontrar consultor</Button>
<Button variant="secondary">Ver perfil</Button>
<Button variant="ghost" iconRight={<ChevronRight size={16} />}>Saber mais</Button>
```

Variants: `primary` (calm gold gradient, navy ink — one per view), `secondary` (gold-hairline outline), `ghost` (quiet text). Sizes `sm | md | lg`. Pass `onIvory` when placed on the ivory light section. Use the BRIGHT title gold for headings/prices — never for buttons; buttons always use the calmer gold.
