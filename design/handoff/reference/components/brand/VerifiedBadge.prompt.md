The in-product trust mark for approved consultants. The pill sits inline beside an agent's name; the seal (Concept B) anchors a profile header.

```jsx
<VerifiedBadge variant="pill" label="Verificado" />
<VerifiedBadge variant="seal" sealSize={72} />
```

The verified green (`--green-verified`, #3FB984) appears ONLY here and on other genuine success states — never as a decorative accent. Show this badge only on agents the platform has actually approved; honesty about what is verified vs. preview is core to the brand.
